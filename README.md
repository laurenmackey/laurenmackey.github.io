# laurenmackey.github.io
Personal Website

## Setup
Delete main local directory

Clone
    git clone https://github.com/laurenmackey/laurenmackey.github.io.git
    
Install bundler

    sudo gem install bundle
    
Install all the ruby gems

    bundle install
    
Install all the node modules

    nvm use .
    npm install
    npm install -g gulp
    
## Run

    gulp
